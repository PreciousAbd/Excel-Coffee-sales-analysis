Coffee Sales Analysis Dashboard (Excel)

  PROJECT OVERVIEW

This project showcases an end-to-end Excel data analytics workflow, transforming raw coffee sales data into an interactive dashboard that delivers clear business insights.

The goal of the project is to demonstrate practical skills in data cleaning, analysis, and visualization using Microsoft Excel.

TOOLS USED

. Microsoft Excel

. Pivot Tables

. Pivot Charts

. Excel Functions (IF, XLOOKUP/VLOOKUP, TEXT, etc.)

. Slicers & Interactive Filters
